# math_5_checkup_tasks
Математический тренажёр для 5 класса – интерактивное приложение на Python с GUI (PyQt5), помогающее ученикам отрабатывать ключевые темы для подготовки к МЦКО. Приложение генерирует случайные задания с проверкой ответов и виртуальной клавиатурой. Тэги: python, pyqt5, math-trainer, education, desktop-application, mathematics, 5th-grade
